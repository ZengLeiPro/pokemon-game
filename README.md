# 关都文字对战：初始冒险

一个基于浏览器的宝可梦文字对战游戏，使用纯HTML、CSS和JavaScript实现。

## 游戏特色

- 🎮 **回合制战斗系统**：经典的1v1对战玩法
- 🔥 **御三家选择**：小火龙、杰尼龟、妙蛙种子任你选择
- ⚡ **属性克制系统**：火克草、水克火、草克水等完整克制关系
- 📈 **成长系统**：经验获取、等级提升、宝可梦进化
- 💾 **自动保存**：游戏进度自动保存到浏览器本地存储
- 📱 **跨端兼容**：支持电脑、平板、手机浏览器

## 快速开始

### 方法1：直接打开
1. 下载或克隆这个仓库
2. 双击打开 `index.html` 文件
3. 开始游戏！

### 方法2：使用本地服务器
```bash
# 如果你有Python 3
python -m http.server 8000

# 或者使用Node.js的http-server
npx http-server

# 然后在浏览器访问 http://localhost:8000
```

## 游戏玩法

### 开始游戏
1. 选择你的初始宝可梦（小火龙/杰尼龟/妙蛙种子）
2. 点击"开始战斗"遇到野生宝可梦
3. 选择技能进行对战

### 战斗系统
- **攻击技能**：造成伤害，受属性克制影响
- **辅助技能**：提升己方属性或降低对方属性
- **属性克制**：
  - 火系 克制 草系（2倍伤害）
  - 水系 克制 火系（2倍伤害）
  - 草系 克制 水系（2倍伤害）
  - 电系 克制 水系、飞行系（2倍伤害）
  - 电系对地面系无效（0倍伤害）

### 成长系统
- 战胜野生宝可梦获得经验值（等级 × 5）
- 累积经验值可以升级
- 升级时提升HP、攻击、防御属性
- 达到特定等级自动进化：
  - 小火龙 → 16级进化成火恐龙 → 36级进化成喷火龙
  - 杰尼龟 → 16级进化成卡咪龟 → 36级进化成水箭龟
  - 妙蛙种子 → 16级进化成妙蛙草 → 36级进化成妙蛙花

### 其他功能
- **自动回血**：每场战斗结束后HP自动恢复至满
- **宝可梦中心**：随时手动恢复HP
- **战绩统计**：记录胜场和总场数
- **本地保存**：游戏进度自动保存

## 技术栈

- **前端**：纯HTML5 + CSS3 + JavaScript (ES6)
- **存储**：localStorage（浏览器本地存储）
- **无需后端**：完全离线可玩

## 项目结构

```

├── index.html              # 游戏主页面
├── styles.css              # 样式文件
├── scripts/
│   ├── data.js            # 游戏数据（宝可梦、技能、属性克制表）
│   ├── move.js            # Move类
│   ├── pokemon.js         # Pokemon类
│   ├── battle.js          # Battle类和AI系统
│   ├── gameState.js       # 游戏状态管理
│   ├── ui.js              # UI控制模块
│   └── main.js            # 主程序入口
├── assets/                 # 资源文件夹（可选）
└── README.md              # 本文件
```

## 游戏数据

### 已实现的宝可梦
- **御三家**：小火龙系列（3只）、杰尼龟系列（3只）、妙蛙种子系列（3只）
- **野生宝可梦**：绿毛虫、独角虫、小拉达、波波、皮卡丘、穿山鼠

### 技能系统
- 攻击技能：火花、水枪、藤鞭、电击、撞击、啄等
- 辅助技能：摇尾巴（降防御）、缩入壳中（升防御）、瞪眼（降防御）
